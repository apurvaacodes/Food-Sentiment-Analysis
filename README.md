🍔 **Food Sentiment Analysis - Project Summary**
Food Sentiment Analysis is an intelligent web application that leverages machine learning and natural language processing to automatically analyze food reviews and predict customer sentiment. The application classifies reviews as either Positive or Negative with high accuracy (~95%+), providing confidence scores for each prediction.

What is it?
A Streamlit-based web application that uses Logistic Regression and TF-IDF vectorization to perform real-time sentiment analysis on food reviews. It enables businesses to understand customer feedback at scale, identifying satisfaction levels and trends from customer reviews.

Who is it for?
🏪 Restaurant owners and managers
🍕 Food delivery platforms (Zomato, Uber Eats, DoorDash)
📊 Market researchers and data analysts
🏢 Hotel and hospitality businesses
👨‍💼 Business intelligence professionals
Why was it built?
Manually reading and analyzing hundreds of customer reviews is time-consuming and subjective. This project automates the process, providing:

**How It Works**

Simple Workflow

User Input (Review)
    ↓
Text Preprocessing
    ↓
TF-IDF Vectorization
    ↓
Logistic Regression Model
    ↓
Sentiment Prediction (Positive/Negative)
    ↓
Confidence Score Calculation
    ↓
Display Results & Store in History



**Step-by-Step Process**

**Input**: User enters a food review or uploads a CSV file
**Preprocessing**: Text is cleaned and standardized
**Vectorization**: TF-IDF converts text to numerical features
**Prediction**: Logistic Regression model classifies sentiment
**Scoring**: Confidence score indicates prediction reliability
**Output**: Results displayed with visualizations
**Storage**: Predictions saved to session history
